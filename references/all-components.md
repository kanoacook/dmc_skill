## How to Use This Reference

1. **Look up your component** in the list above
2. **Check the prop name** — it must be camelCase exactly as listed
3. **Verify no typos** — common mistakes like `icon=` instead of `leftSection=` are documented in `common-mistakes.md`
4. **Remember shared props** — spacing/layout props (m, p, c, bg, w, h) work on all components
5. **For callbacks** — components accepting `n_clicks`, `n_blur`, `n_submit`, or `value` need `id=` and require Dash callbacks

---

## Key Points

- ✅ **camelCase** — all props use camelCase, not snake_case
- ✅ **No JS handlers** — no `onChange`, `onClick`, `onFocus`; use Dash callbacks instead
- ✅ **Callback components need `id=`** — anything read/written in a callback must have an id
- ✅ **Children first** — positional first arg or `children=` kwarg for nested content
- ✅ **Exact prop names matter** — hallucinated props cause runtime AttributeError
